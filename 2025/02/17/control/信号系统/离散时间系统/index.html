描述：线性时不变系统的差分方程模型

1. 序列（离散的时间）

(1) 变化

  • 压缩：kn，例如 2n，则只保留原来的偶数项，原来的奇数项为 0。

(2) 典型案例

  1. 单位样值信号 δ(n)
    只在 n = 0 时为 1（而不是 ∞）。

  2. 单位跃变信号 u(n)
    在 n = 0 时取 1（而不是连续系统中 t = 0 时不取值）。

  3. 单位矩阵信号 RN(n)
    定义在区间 0 ≤ n ≤ N − 1。
    > PS：(1)(2)(3) 之间的转化关系见书 p7。

  4. 斜变信号 x(n) = n u(n)

  5. 指数信号 x(n) = anu(n)
    对 a 的取值进行分类讨论，注意在 a < 0 时，x(n) 的取值在正负之间摆动。

  6. 正弦信号 x(n) = sin (ωnn)
    一个周期的采样点数为：

    • 当 N 为整数时，序列严格周期；
    • 当 N 不是整数时，序列仍有周期性，但周期大于 N；
    • 如果无法满足，则序列失去周期性。

方程式表达

采用后向形式表达： y(n) = y(n − 1) + ⋯

求解

1. 迭代法

例如： y(n) − y(n − 1) = 1 假如知道了 y(0)，就可迭代出 y(1), y(2), …。

2. 时域法（求解差分方程）

（1）目标

求 y，即系统的响应。

（2）步骤

  • 求齐次解
  • 求特解
  • 代值求系数
  • 相加得完全解

PS：注意若出现重根如何求解。
特解的形式：主要根据激励函数 x(n) 的形式来确定。若不够条件，可以直接利用差分方程递归求值（例如由 y(0) → y(1) → y(2)）。

（3）响应的组成

  • 激励信号：可以看作输入系统的自变量 x(n)。
  • 响应的线性性： f(x1 + x2) = f(x1) + f(x2)
  • 完全解 = 零输入响应 + 零状态响应
    • 零输入响应：激励信号 x(n) = 0 时系统的响应（即齐次解）。
    • 零状态响应：考虑激励信号 x(n) 引起的系统响应，假设初始状态为零。

完全解的产生可以看作逐步累积的过程（书中 p20, p23 有例子）。

（4）特例：单位样值响应 δ(n) → h(n)

  • 系数的确定：令差分方程中的 n = 0，得到 h(0)，并且规定 h(−1) = h(−2) = ⋯ = 0。
  • 零输入响应：在此基础上继续求解（见书 p28）。

作用： 1. 若 h(n) = 0 对 n < 0 成立，则系统为因果系统。
2. 若 h(n) 收敛，则系统稳定。
